# MMP.AI-assets

A public library of AI-generated images. Built primarily for MMPWorks projects (Herald, mmpworks.com, presentations, docs) — free for anyone to use.

## License

Everything here is [CC-BY 4.0](LICENSE). Use the images anywhere — commercial or not — with attribution:

> Images from [mmpworks/MMP.AI-assets](https://github.com/mmpworks/MMP.AI-assets), CC-BY 4.0.

A note on AI-generated work: copyright in purely AI-generated images is unsettled in some jurisdictions. Where an image carries no copyright, the license simply doesn't restrict you further — you still get everything CC-BY grants, at minimum.

## Layout

```
images/
  <collection>/        one folder per theme or project
    <name>.png
    <name>.md          optional: prompt, model, date, notes
```

Each collection may include a short manifest describing what the images are and how they were made (model, prompt, post-processing).

## Contributing

This is primarily an MMPWorks-curated library. Issues and suggestions are welcome; image submissions are accepted at the maintainers' discretion and must be releasable under CC-BY 4.0.
